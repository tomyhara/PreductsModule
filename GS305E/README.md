# GS305E Mount

## About GS305E mount

[GS305](https://www.netgear.com/jp/business/wired/switches/plus/gs305e/)はNETGEAR製のスイッチングハブです。

これ以外にもNETGEAR製の5ポートのスイッチングハブでしたら使用できるかと思います。

## 使用方法

stlデータを2個(1つは反転させたもの)印刷してください。

Base SliderにM5x10mmの六角穴付きボルトで固定します。
